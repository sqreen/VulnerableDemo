# How to use this application


## Setup

This is a regular Rails application, setup it as you would do with any other
application.

If you don't have bundler, install it:

    $ sudo gem install bundler

Once bundler is installed:

    $ bundle
    $ rake db:setup

### Run 

    $ rails s
